CSS Properties and Their Usage

color

Sets the color of text. Example: color: blue;

background-color

Sets the background color of an element. Example: background-color: #FF0000;

font-size

Sets the size of the font. Example: font-size: 16px;

font-family

Specifies the font for text. Example: font-family: Arial, sans-serif;

margin

Sets the margin around elements. Example: margin: 10px;

padding

Sets the padding inside elements. Example: padding: 15px;

border

Sets the border around elements. Example: border: 1px solid black;

width

Sets the width of an element. Example: width: 100px;

height

Sets the height of an element. Example: height: 200px;

display

Specifies how an element should be displayed. Example: display: flex;

position

Specifies the positioning method. Example: position: relative;

text-align

Sets the horizontal alignment of text. Example: text-align: center;

text-decoration

Sets or removes decorations from text. Example: text-decoration: underline;

float

Specifies how an element should float. Example: float: left;

opacity

Sets the opacity level of an element. Example: opacity: 0.5;

flex-direction

Sets the direction of flex items in a container. Example: flex-direction: row;

justify-content

Aligns flex items along the main axis. Example: justify-content: space-between;

align-items

Aligns flex items along the cross axis. Example: align-items: center;

flex-wrap

Specifies whether flex items should wrap. Example: flex-wrap: wrap;

grid-template-columns

Defines columns in a grid layout. Example: grid-template-columns: 1fr 1fr 1fr;

grid-template-rows

Defines rows in a grid layout. Example: grid-template-rows: 100px auto;

gap

Sets the gaps between grid/flex items. Example: gap: 20px;

grid-column

Specifies an item's column position in grid. Example: grid-column: 1 / 3;

grid-row

Specifies an item's row position in grid. Example: grid-row: 2 / 4;

flex-grow

Sets the growth factor of a flex item. Example: flex-grow: 1;
