HTML Basics

What is HTML?

HTML (HyperText Markup Language) is the foundation of web development. It is used to structure content on web pages using elements and tags.

Basic HTML Commands

Below are some common HTML tags with examples:

1. Headings

Headings are defined using <h1> to <h6>. Here's an example:

<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>

Result:

This is a Heading 1

This is a Heading 2

2. Paragraphs

Paragraphs are created using the <p> tag:

<p>This is a paragraph of text.</p>

Result:

This is a paragraph of text.

3. Links

Hyperlinks are created using the <a> tag:

<a href="https://www.example.com">Visit Example</a>

Result:

Visit Example

4. Images

Images are added using the <img> tag:

<img src="example.jpg" alt="Example Image" width="300" height="200">

Result:

5. Lists

You can create ordered and unordered lists:

Ordered List:

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
  1. Item 1
  2. Item 2

Unordered List:

<ul>
  <li>Item A</li>
  <li>Item B</li>
</ul>
  • Item A
  • Item B

6. Tables

Tables are created using the <table> tag:

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>25</td>
  </tr>
</table>

Result:

Name Age Alice 25

7. Forms

Forms collect user input:

<form>
  <label>Name: <input type="text" name="name"></label>
  <button type="submit">Submit</button>
</form>

Result:

8. Audio

Audio is added using the <audio> tag:

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

Result:

9. Video

Video is added using the <video> tag:

<video controls width="320" height="240">
  <source src="video.mp4" type="video/mp4">
  Your browser does not support the video tag.
